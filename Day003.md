## Statement
```
Write a program that greets the user by printing the word "Hello", a comma, the name of the user and an exclamation mark after it. 
```
## Sample input
```
Harry
```
## Sample output
```
Hello, Harry!
```
## code
```
# Read a string:
a = str(input())

# Print a string:
print("Hello"+", "+a+"!")

```
