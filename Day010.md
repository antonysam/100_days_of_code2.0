## Statement
```
Given an integer, print "odd" if it's odd and print "even" otherwise.

```
## Sample input1
```
5
```
## Sample output1
```
odd
```
## Sample input2
```
6
```
## Sample output2
```
even
```
## code
```
# Read an integer:
# a = int(input())
# Print a value:
# print(a)
num = int(input())
if(num%2==0):
  print("even")
else:
    print("odd")
```
