## Statement
```
Write a program that takes three numbers and prints their sum. Every number is given on a 
separate line.
```
## Sample input
```
2
3
6
```
## Sample output
```
11
```
## code
```
# Getting inputs from user

a = int(input())
b = int(input())
c = int(input())

# Printing the result

print(a + b+c)
```
